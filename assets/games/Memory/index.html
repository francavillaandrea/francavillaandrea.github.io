Come si gioca a Memory
Obiettivo:

Trova tutte le coppie di carte uguali nel minor numero di mosse e nel minor tempo possibile.

Come giocare:
  1. Clicca su due carte per scoprirle.
  2. Se le carte sono uguali, rimarranno scoperte.
  3. Se le carte sono diverse, si rigireranno dopo un breve periodo.
  4. Continua fino a quando tutte le coppie non saranno state trovate.
Difficoltà:

Scegli una difficoltà per cambiare la dimensione della griglia e il numero di coppie da trovare.
